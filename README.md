# Street-Level Planning Intelligence

A research project on street-level perception, planning intelligence, and multimodal graph learning for healthy community analysis in Singapore.

## Overview
This project investigates how street-level perception can be integrated into planning-oriented urban analysis through multimodal graph learning. It focuses on reimagining healthy communities in Singapore by combining street-level representations, urban spatial data, and graph-based modeling.

## Motivation
Many planning and urban analytics workflows rely heavily on aggregated or top-down indicators, while street-level conditions and perceptual qualities remain underrepresented. This project explores how machine learning can better connect street-level urban experience with planning intelligence.

## Current Status
This project is part of an ongoing research trajectory in urban AI and spatial computing.

## Repository Availability
The full research code and data pipeline are not fully public at this stage. This repository serves as a public-facing overview of the project.
